# MedConnect Backend

This is the Node.js backend for the MedConnect application.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the server in development mode:
   ```
   npm run dev
   ```

3. Or run in production mode:
   ```
   npm start
   ```

The server will run on port 3000 by default.

## API Endpoints

- GET / : Basic hello message
- POST /api/auth/login : Login endpoint (placeholder)
- POST /api/auth/register : Register endpoint (placeholder)
